- 사용자 행동(view_reflection, write_reflection)을 정확히 기록하고

## 🔥 1. view_reflection 처리

### ✔ 발생 조건
- 사용자가 "나눔(묵상)"을 클릭하여 읽을 때
- 반드시 명시적 클릭 기반

---

### ✔ 처리 방식
- 대표 구절(mainVerseId) 1개만 적용

---

### 코드

```javascript
function handleViewReflection(userId, mainVerseId) {
  if (!canUpdate(userId, mainVerseId, "view_reflection")) return

  const verse = findVerse(mainVerseId)

  verse.score += 2
  verse.recentScore += 2

  saveVerse(verse)
  saveActionLog(userId, mainVerseId, "view_reflection")
}
```

## 🔥 2. write_reflection 처리

### ✔ 발생 조건
- 사용자가 묵상을 작성하고 저장할 때

---

### ✔ 처리 방식
- 여러 절 선택 가능
- 총 점수(3)를 절 수로 나누어 분배

---

### 코드

```javascript
function handleWriteReflection(userId, verseIds) {
  const totalScore = 3
  const perScore = totalScore / verseIds.length

  for (const verseId of verseIds) {
    if (!canUpdate(userId, verseId, "write_reflection")) continue

    const verse = findVerse(verseId)

    verse.score += perScore
    verse.recentScore += perScore

    saveVerse(verse)
    saveActionLog(userId, verseId, "write_reflection")
  }
}
```

## 🔥 3. 주제별보기 조회 알고리즘

### 입력
- category
- N (표시 개수)
- userId
- seenVerseIds

---

### 1. 주제 필터

```javascript
const pool = verses.filter(v => v.mainCategory === category)
```

---

### 2. 최근 본 구절 제외

```javascript
let filtered = pool.filter(v => !seenVerseIds.includes(v.verseId))
if (filtered.length < N) filtered = pool
```

---

### 3. 대표 구절 선택

```javascript
const anchors = filtered.filter(v => v.isAnchor)
const mainPick = weightedRandom(anchors.length ? anchors : filtered)
```

---

### 4. 나머지 선택

```javascript
let candidates = filtered.filter(v => v.verseId !== mainPick.verseId)

const result = [mainPick]

while (result.length < N && candidates.length > 0) {
  const pick = weightedRandom(candidates)
  result.push(pick)
  candidates = candidates.filter(v => v.verseId !== pick.verseId)
}
```

---

### 5. 가중치 계산

```javascript
function getFinalWeight(v) {
  return v.baseWeight + v.score + v.recentScore
}
```

---

### 6. weightedRandom

```javascript
function weightedRandom(items) {
  const total = items.reduce((sum, i) => sum + getFinalWeight(i), 0)
  let rand = Math.random() * total

  for (const item of items) {
    rand -= getFinalWeight(item)
    if (rand <= 0) return item
  }

  return items[items.length - 1]
}
```

function getAll(category) {
  return verses
    .filter(v => v.mainCategory === category)
    .sort((a, b) => {
      if (a.isAnchor !== b.isAnchor) return a.isAnchor ? -1 : 1

      const diff = getFinalWeight(b) - getFinalWeight(a)
      if (diff !== 0) return diff

      return getBibleOrder(a) - getBibleOrder(b)
