# Agent.md – 모줄성 UX & 서비스 구조 정리

## ✅ 결정사항

### 1. 핵심 개념
- 성경을 색으로 읽는 경험 (판단 중심)
- 정답 없음, 사용자 해석 중심
- 읽기 → 색칠 → 반복 구조

### 2. 핵심 UX 원칙
- 설명보다 경험
- 선택 → 색 → 표시 (2클릭 구조)
- 생각을 유도, 정답 제공 금지

### 3. 읽기 구조
- 절이 아닌 ‘의미 단위’로 분리
- 문단 클릭 → 색 선택 → 자동 줄 표시
- 오디오는 보조 기능 (비강조)

### 4. 나누기 구조
- 페이지 캡쳐 ❌
- 템플릿 기반 이미지 자동 생성 ⭕
- 이미지 저장 + 문구 복사
- SNS 업로드는 사용자 영역

### 5. 생각 나누기
- 한 줄 묵상 모음 공간
- 댓글, 좋아요 없음
- 입력은 읽기/써보기에서 발생 → 모여서 보여짐

### 6. 돌아보기
- 주제 분포 (핵심)
- 읽기 기록
- 표시한 구절
- 자기 인식 중심 (점수/비교 없음)

### 7. 함께 써보기
- PDF 워크북 제공
- 종이에 직접 줄치기
- 한 줄 묵상 작성
- 공유는 선택 (사진 → SNS)

### 8. 전체 사용자 흐름
읽기 → 색칠 → (선택)
→ 나누기 (SNS)
→ 돌아보기 (자기 인식)
→ 함께 써보기 (실천)
→ 생각 나누기 (확장)

### 9. SNS 전략
- 자동 이미지 생성
- 해시태그 자동 포함 (#모줄성)
- 자연스럽게 공유 유도 (강제 없음)

## ❗ 미결사항

### 1. 12색 카테고리 정의
- 이름 / 색상 / 의미 기준 확정 필요

### 2. 의미 단위 분리 방식
- 자동화 여부
- 편집 기준 정의 필요

### 3. 이미지 템플릿 디자인
- 스타일 2~3개 결정
- 색 반영 방식 정의

### 4. 데이터 구조
- 색 선택 저장 방식
- 묵상 저장 구조
- 주제 분포 계산 방식

### 5. UX 세부 설계
- 나누기 팝업 UI
- 생각 나누기 카드 디자인
- 돌아보기 그래프 형태
