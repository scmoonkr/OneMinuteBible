# 📖 말씀 펼쳐보기

> 성경을 “목차”가 아니라 “이야기 흐름”으로 탐색합니다  
> 클릭하면 다음 단계로 자연스럽게 이동합니다

# 💡 사용 방식

1. 구약 / 신약 선택  
2. 큰 흐름 선택 (모세오경 등)  
3. 책 선택 (창세기 등)  
4. 이야기 선택 (천지창조 등)  

👉 항상 “한 단계씩” 이동합니다  
👉 구조는 보이지 않고, 흐름만 느껴집니다  

---

# 🔥 핵심

👉 성경을 “책”이 아니라  
👉 “이야기”로 펼쳐봅니다
